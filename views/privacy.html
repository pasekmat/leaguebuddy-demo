1. Introduction

LeagueBuddy is an iOS application developed as part of a master's thesis project. This Privacy Policy explains how we collect, use, and protect your information.

2. Data We Collect

Firebase Authentication

  • Firebase User ID (firebase_uid)
  • Basic authentication metadata

We do not store your Google password.

Riot Games Data (When Implemented)

  • Riot ID
  • PUUID
  • Summoner Name
  • Profile information available via Riot API
  • OAuth tokens (stored securely and only as necessary)

Application Data

  • User-created tier lists
  • Aggregated rankings
  • Associated metadata

Data is stored in PostgreSQL.

Technical Data

Limited technical data such as device type, app version, and error logs may be collected for debugging and stability purposes.
